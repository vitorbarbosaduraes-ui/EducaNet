# EducaNet
EducaNet: Plataforma de organização escolar desenvolvida para a E.E. Papa João Paulo I. Utilizando HTML5 e CSS3 com design responsivo, o projeto centraliza tarefas, avisos e materiais didáticos. O foco é otimizar a rotina dos estudantes através de uma interface intuitiva, aplicando conceitos reais de usabilidade e tecnologia web.
